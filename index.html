Recursos

Conta com recursos como janela modal e busca com URL's amigáveis:

  • Janela modal utilizando boostrap
  • Todas as legendas personalizavéis
  • Suporte a obter informações dos item através de arrays para serem exibidos no modal

Busca automática aos elemento com a classe ".filter" declarada pelo usuário

  • Personalização se o campo é requirido ou não
  • Pode ser usando em Buttons e Submits utilizando o evento click, deve ser indicado a rota para o controller de busca.
  • Pode ser utilizado em inputs text, sobre o evento keydown
  • Obtém todos os itens informados com a classe ".filter", pegando como chave o atributo "name" e o valor o atributo "value".
